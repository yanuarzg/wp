1 Generator Artikel AI

Halaman utama plugin di dashboard WordPress. Admin memasukkan sumber, AI menghasilkan artikel lengkap.

Input Sumber

URL berita, teks mentah, keyword, atau beberapa URL sekaligus. Favicon sumber ditampilkan otomatis.

Mode Penulisan

Standar Jurnalistik HE, Straight News (piramida terbalik), Rewrite Artikel, Target SEO Page 1.

Bahasa

Indonesia dan English. Jika English dipilih, seluruh output (judul, isi, meta, ringkasan, sosmed) berbahasa Inggris.

Pengaturan Konten

Panel collapsible berisi Pilih AI, Gaya Penulisan, Panjang Artikel, dan Bahasa.

Byline

Pilihan "Yang Nulis" dan "Yang Edit" otomatis masuk ke konten artikel.

Pintasan Referensi

Icon Google Trends Indonesia dan Google News untuk mencari topik aktual.

Progress Generate

Lima tahap: membaca sumber, mengunci fakta, menulis artikel, mengambil media, finalisasi draft.

Copy HTML

Tombol menyalin judul H1 dan isi artikel dalam format HTML ke clipboard.

2 Multi AI Provider

Plugin tidak terikat satu penyedia AI. Pengguna bisa memilih provider langsung dari dropdown.

Gemini

Provider utama. Model didukung: Gemini 2.5 Flash, Flash Lite, Pro, dan model lain yang tersedia di dropdown pengaturan.

Grok / xAI

Untuk kebutuhan real-time dan akurasi berbasis informasi terbaru. Badge PRO, perlu akun GROK PRO.

Z.AI

Didukung dengan pola OpenAI-compatible API.

Groq

Provider tambahan (v3.3.10). Link ambil API: console.groq.com/keys, endpoint OpenAI-compatible.

Dropdown Pilih AI

Di halaman generator, admin bisa berganti provider kapan saja tanpa harus ke halaman pengaturan.

Fallback Provider

Jika Grok/xAI gagal, plugin bisa fallback ke Gemini jika tersedia.

3 Pengaturan API Key Aman

Semua API key dikelola dari halaman Pengaturan API. Tidak ada key bawaan di ZIP plugin.

Input Terpisah per Provider

API Gemini, Grok/xAI, Z.AI, dan Groq masing-masing punya input sendiri, satu key per baris.

Tombol Ambil API Key

Setiap provider memiliki tombol menuju halaman resmi pengambilan API key.

Test API Key

Tombol test untuk setiap provider. Hasil test muncul tepat di bawah tombol, bukan di tempat terpisah.

Masking Key

API key ditampilkan tersensor (contoh: AIzaSyA••••••••••). Key diblokir disimpan sebagai hash, bukan teks asli.

Auto-block Key Bermasalah

Key yang bocor, invalid, expired, disabled, forbidden (401/403) otomatis ditandai dan dilewati.

Rotasi & Retry

Jika satu key limit/timeout/high demand/5xx, plugin coba key berikutnya. Retry beberapa kali.

Prompt Custom

Bagian Instruksi Global Tambahan. Admin bisa menambahkan instruksi khusus tanpa mengedit kode plugin.

4 Source Lock

Fitur kunci untuk menjaga akurasi fakta dari artikel sumber.

Ekstraksi Teks Sumber

Jika input berupa URL, plugin membaca isi artikel langsung dari sumber via server WordPress, bukan membiarkan AI mengambil informasi bebas.

Data yang Dikunci

Tanggal, jam, nama narasumber, jabatan, angka, lokasi, status resmi, dan kutipan langsung.

Kutipan Langsung

Kutipan dalam tanda petik tidak boleh diubah sama sekali (verbatim).

AI hanya diberi teks sumber hasil ekstraksi, bukan dibiarkan mengambil informasi bebas dari internet.
5 Safe Paraphrase

Menyeimbangkan akurasi sumber dan keunikan redaksional.

Fakta Tetap Sama

Tanggal, angka, nama, lokasi, dan status resmi tidak berubah.

Kutipan Tetap Verbatim

Kutipan langsung tidak diparafrase.

Narasi Diparafrase

Struktur kalimat dibuat lebih unik, urutan penjelasan bisa disusun ulang secara aman.

Deteksi Kemiripan

Jika kemiripan narasi >50%, status dianggap tinggi dan tersedia opsi parafrase ulang. Kutipan dikecualikan dari perhitungan.

Compare

Tombol membandingkan teks sumber dan hasil rewrite dalam tampilan split.

6 Audit Fakta & Quality Score

Fitur kontrol kualitas sebelum artikel masuk Draft WordPress.

Panel Audit Fakta

Memeriksa: kutipan di sumber, tanggal di sumber, jam di sumber, angka di sumber, URL sumber, caption istimewa/ist. Item yang aman disembunyikan.

Normalisasi Tanggal

24/05/2026, 24 Mei 2026, 24 May 2026, May 24, 2026 dianggap sama. Mengurangi false warning.

Quality Score

Skor berdasarkan: sumber berhasil dibaca, data cocok, SEO lengkap, panjang, heading, blockquote, list, kemiripan narasi.

Deteksi Duplikasi

Jika kemiripan >50%, status tinggi dan ada opsi parafrase ulang.

7 SEO Metadata Lengkap

Plugin menghasilkan pilihan SEO title, slug, meta description, tags, dan kategori.

Pilihan SEO Title & Slug

Beberapa opsi dengan radio button. Title terpilih jadi H1 dan judul draft.

Meta Description

Beberapa opsi. Disimpan ke Yoast, Rank Math, SEOPress, fallback custom meta.

Post Subtitle

Mendukung field single_post_subtitle dan _single_post_subtitle. Meta description otomatis jadi subtitle.

Tags & Kategori Otomatis

Tags relevan dan rekomendasi kategori berdasarkan isi artikel.

8 Manajemen Gambar (v3.3.10)

Fitur pengambilan, pemilihan, dan optimasi gambar dari artikel sumber.

Deteksi Gambar Sumber

Mengambil gambar dari area artikel, bukan icon/logo kecil. Minimal lebar ~320px, deduplikasi gambar mirip.

Multi-select Gambar

Checklist untuk memilih beberapa gambar. Urutan pertama = featured image, berikutnya = masuk konten.

Caption & Credit

Deteksi caption dari figcaption, class caption, alt, data-caption, dll. Credit dari source/photo-credit. Audit caption istimewa/ist.

Auto Compress & WebP

Resize jika >1280px, kompres kualitas 82, konversi ke WebP. Fallback ke asli jika gagal. Metadata disimpan.

Featured Image Otomatis

Download ke Media Library, set featured image, sisipkan gambar ke konten sebagai figure.

9 Kirim ke Draft WordPress

Satu klik mengirim artikel lengkap ke Draft WordPress.

Data yang Dikirim

Title, slug, content, excerpt, category, tags, meta description, post subtitle, source URL, AI provider, versi plugin, gambar.

Gambar di Draft

Gambar pertama jadi featured image, berikutnya disisipkan ke konten sebagai figure dengan caption.

Metadata Tersimpan

URL gambar asli, status optimasi, format hasil (WebP/fallback/original).

10 Distribusi Konten

Satu artikel bisa langsung dikembangkan menjadi berbagai format.

Ringkasan Cepat

Poin-poin utama artikel untuk editor, bullet summary, atau bahan caption singkat.

Draft Sosial Media

Twitter/X thread, caption Instagram/Facebook, hashtag relevan, CTA diskusi.

Skrip Video Pendek

Ide visual/B-roll, hook 0–3 detik, isi voiceover, CTA. Untuk TikTok/Reels/Shorts.

20 Riwayat + Pagination

Multi URL

Indikator Status

12 Directory Subdomain (Network)

Hanya muncul di harianexpress.com dan subdomainnya. Membaca data dari file eksternal.

Sumber Data

Directory A-Z

Pencarian, filter grup, sortir A-Z, terbaru, terlama, artikel terbanyak, tersedikit. Navigasi alfabet.

Subdomain Utama

www.harianexpress.com di posisi paling atas dengan label "Utama".

Artikel Terbaru

Judul terbaru, total artikel, waktu publish (time ago), kategori dari REST API.

Status Warna

<1 minggu 1–4 minggu >1 bulan

Shortcut

Icon buka website di tab baru dan login ke /pinarak.php.

Domain Bawaan

15 Autopilot Engine (v5.0)

Sistem baru AI Journalist yang bekerja sepenuhnya otomatis tanpa input manual sumber.

Full Automation Mode

Sistem tidak lagi membutuhkan input URL manual. AI memilih sumber berita secara otomatis.

Smart Source Selection

Google News, RSS global, dan feed trend dipilih berdasarkan kualitas dan validitas sumber.

AI Decision Layer

AI menentukan apakah berita layak diproses berdasarkan nilai berita (news value).

16 Multi-Source Intelligence Engine

Sistem korelasi berita dari berbagai sumber untuk menghasilkan artikel yang lebih akurat dan kontekstual.

Korelasi Multi Sumber

AI membandingkan beberapa sumber berita sebelum menentukan angle artikel.

Source Neutral Processing

Tidak ada sumber utama. Semua sumber diperlakukan setara untuk menghindari bias urutan.

Conflict Detection

Sistem mendeteksi jika ada perbedaan fakta antar sumber dan menyusunnya dalam narasi netral.

17 Master Scheduler System

Sistem penjadwalan inti yang mengatur semua proses auto publishing.

Master Cron Loop

Berjalan setiap 5 menit untuk mengecek task tanpa tergantung WP-Cron sepenuhnya.

Slot-Based Publishing

Setiap interval hanya menghasilkan 1 artikel untuk mencegah overload dan duplikasi.

Auto Recovery

Sistem otomatis memperbaiki jadwal jika terjadi delay atau cron gagal berjalan.

18 Global Anti-Duplicate Guard

Sistem pencegahan duplikasi artikel di seluruh jaringan website.

Cross-Site Detection

Artikel dicek bukan hanya di satu website, tetapi di seluruh network.

Similarity Engine

Judul dan isi artikel dibandingkan untuk mendeteksi konten mirip sebelum publish.

Smart Skip Logic

Jika semua kandidat sudah pernah dipublish, sistem otomatis skip tanpa error.

19 Slot Lock Publishing System

Menghindari double publish dalam satu interval waktu.

1 Slot = 1 Artikel

Setiap jadwal hanya boleh menghasilkan satu artikel meskipun ada banyak sumber.

Lock Engine

Setelah satu artikel berhasil dibuat, slot langsung dikunci sampai interval berikutnya.

Conflict Prevention

Mencegah Google Trends dan Global Trends berjalan bersamaan dalam satu cycle.

20 Feed Intelligence Scoring

Sistem penilaian otomatis untuk memilih berita terbaik.

Relevance Score

Menilai relevansi berita terhadap tren global atau lokal.

Freshness Score

Berita terbaru mendapat prioritas lebih tinggi.

Image Quality Score

Sumber dengan gambar valid dan berkualitas lebih diprioritaskan.

21 Content Network Distribution v5

Sistem distribusi artikel antar website dalam satu jaringan HE Network.

Instant Sync

Artikel yang dipublish langsung dikirim ke aggregator tanpa delay.

Smart Ping System

Ping hanya dilakukan jika artikel valid dan lolos semua filter.

Cache Invalidation

Cache antar website dalam network otomatis diperbarui saat artikel baru masuk.

22 System Evolution (v3 → v5)

Perubahan besar arsitektur AI Journalist dari sistem manual ke autopilot penuh.

Dari Manual ke Autopilot

Sistem berkembang dari generator artikel berbasis input menjadi autonomous news engine.

Dari Single Source ke Multi Source

AI tidak lagi bergantung pada satu URL, tetapi menganalisis banyak sumber sekaligus.

Dari Cron Biasa ke Master Scheduler

Sistem penjadwalan sekarang menggunakan slot-based engine yang lebih stabil.

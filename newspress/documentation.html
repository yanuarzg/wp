NewsPress WordPress Theme

NewsPress adalah theme WordPress untuk portal berita dan blog editorial. Theme ini menyediakan layout headline, breaking news, widget homepage, sidebar, slot iklan, share buttons, breadcrumbs, related posts, table of contents, schema Article/NewsArticle, SEO panel, demo import, lisensi lokal, dan updater GitHub.

Dokumentasi ini disusun berdasarkan paket theme NewsPress 1.6.3. Ganti link download, link demo, harga, dan kontak support sesuai kebutuhan website Anda.

1. Instalasi Theme

  1. Masuk ke dashboard WordPress.
  2. Buka Appearance → Themes → Add New → Upload Theme.
  3. Upload file newspress.zip, lalu klik Install Now.
  4. Setelah selesai, klik Activate.
  5. Buka Appearance → Menus, buat menu, lalu assign ke Primary Menu dan/atau Footer Menu.
  6. Buka Appearance → Customize untuk mengatur logo, warna, layout, breaking news, slot iklan, SEO, dan performa.

2. Homepage Builder

Homepage Builder berada di Appearance → Homepage Builder. Fitur ini mengatur struktur row homepage, jumlah artikel, dan navigasi latest posts.

Posts per rowJumlah artikel dalam tiap row, maksimal 10 artikel.
Homepage TopMenampilkan widget area di atas hero.

Row ad enabledMenampilkan slot iklan sebelum latest posts pada setiap row.

Section bawaan homepage

NewsPress mengenali section hero, category_grid, category_list, ad, dan widget. Tampilan visualnya dapat diperkaya melalui widget area.

3. Pengaturan Customizer

NewsPress Design

Mengatur warna utama, warna aksen, warna teks, warna mode gelap, dan layout default. Layout tersedia: Right Sidebar, Left Sidebar, dan No Sidebar.

NewsPress Options

Mengatur jam realtime, breaking ticker speed, trending source, trending range, jumlah item trending, format tanggal artikel, published/updated date, view count, label view, footer bottom custom text, dark/light toggle, default color scheme, sticky sidebar, dan share buttons.

NewsPress SEO

Mengaktifkan fallback meta description, Open Graph, Twitter Card, JSON-LD schema, auto-disable schema saat plugin SEO besar aktif, preload featured image artikel, dan preconnect Google Fonts.

NewsPress Performance

Mengatur optimasi frontend yang aman untuk PageSpeed, seperti disable emoji, disable embeds, clean head, remove block CSS, remove global styles, remove Dashicons untuk pengunjung, defer theme JS, dan preload homepage hero image.

5. Breaking News dan Trending

Breaking News dapat mengambil sumber dari tag breaking-news, kategori terpilih, atau recent posts. Jika menggunakan mode tag, buat tag dengan slug breaking-news dan tempelkan pada artikel yang ingin muncul di ticker.

Trending Topics dapat diambil dari tag atau kategori. Rentang waktunya dapat diatur ke All time, Today, Last 7 days, atau Last 30 days.

7. Halaman Artikel

Single post NewsPress mendukung featured image dengan caption, breadcrumbs, meta author/date/reading time/view count, share rail, table of contents otomatis, tag, author profile, related posts, komentar, dan navigasi post dengan thumbnail.

Di editor post, gunakan NewsPress SEO Panel untuk mengatur meta description, tampilan meta description di bawah judul, schema type, table of contents, dan author profile box.

8. SEO dan Schema

NewsPress menyediakan SEO fallback untuk situs yang belum menggunakan plugin SEO khusus. Fitur yang tersedia meliputi meta description fallback, tag keywords, Open Graph, Twitter Card, JSON-LD schema Article/NewsArticle, dan breadcrumb schema.

Jika situs memakai plugin SEO utama, aktifkan opsi Disable theme JSON-LD when a major SEO plugin is active agar tidak terjadi duplikasi schema.

9. Performance

NewsPress memakai CSS Grid/Flexbox dan vanilla JavaScript. Opsi performa bawaan dapat membantu mengurangi asset frontend yang tidak diperlukan.

  • Disable WordPress emoji script/styles.
  • Disable wp-embed script.
  • Remove unused legacy head links.
  • Remove default block library CSS pada frontend.
  • Remove global styles inline CSS/SVG filters.
  • Remove Dashicons untuk pengunjung yang tidak login.
  • Defer NewsPress frontend JavaScript.
  • Preload homepage hero image.
  • Lazy loading dan decoding async untuk gambar konten/avatar.

10. Demo Import

Theme menyertakan file demo demo/default-import.json. Fitur demo import/export dapat membawa pengaturan kategori, posts, featured images, widget settings, Customizer settings, menu, dan homepage sections. Pastikan lisensi aktif jika fitur demo import dikunci oleh sistem lisensi.

11. Lisensi dan Update

NewsPress memiliki halaman Appearance → NewsPress License. Sistem lisensi mendukung validasi lokal untuk annual dan lifetime license, termasuk mode wildcard subdomain. Theme updater membaca manifest update dari repository GitHub yang telah dikonfigurasi di theme.

Halaman lisensi juga menyediakan informasi diagnostik dan tombol WA Support. Gunakan diagnostic copy saat pengguna melaporkan masalah aktivasi atau update.

12. FAQ Singkat

Kenapa Breaking News tidak muncul?

Pastikan ada artikel dengan tag slug breaking-news, atau ubah sumber breaking ke kategori/recent posts di Customizer.

Kenapa related posts kosong?

Pastikan artikel memiliki kategori/tag yang relevan dan jumlah post cukup. Atur jumlah related posts dari Customizer.

Apakah perlu plugin SEO?

Tidak wajib. NewsPress memiliki SEO fallback. Namun untuk kebutuhan SEO lanjutan, plugin SEO tetap bisa dipakai. Aktifkan opsi disable schema theme agar tidak duplikat.

Apakah theme memakai Bootstrap atau jQuery?

Tidak untuk frontend. Theme memakai native CSS Grid/Flexbox dan vanilla JavaScript.

Bagaimana menampilkan iklan tengah artikel?

Isi slot Article Middle di Customizer atau gunakan widget area Article Ad: Middle Article, lalu tentukan paragraf penyisipan melalui opsi Middle article ad after paragraph.

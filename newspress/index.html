Dirancang khusus untuk kebutuhan media online.

Setiap bagian dibuat agar redaksi, blogger, maupun pemilik portal berita bisa mengatur layout, konten, dan monetisasi langsung dari dashboard WordPress.

Homepage fleksibel

Susun halaman depan seperti portal berita profesional.

NewsPress memberi kontrol terhadap alur konten di homepage: mulai dari headline, kategori pilihan, latest posts bertahap, widget kanan, sampai slot iklan per baris.

  • Jumlah main row dapat diatur dari 1 sampai 8.
  • Jumlah artikel per row dapat diatur dari 1 sampai 10.
  • Latest posts mendukung Load More atau Pagination.
  • Widget area tersedia untuk hero, sidebar homepage, row ad, dan footer.
  • Section bawaan mencakup hero, category grid, category list, ad, dan widget.

SEO & artikel

Lebih siap untuk mesin pencari dan pembaca.

NewsPress menambahkan panel SEO ringan, metadata sosial, schema, serta komponen artikel yang membantu pengalaman membaca.

SEO Panel di editor

Penulis dapat mengisi meta description, melihat Google Preview, memilih schema NewsArticle atau Article, serta mengatur author box dan daftar isi per artikel.

Open Graph & Twitter Card

Theme dapat menghasilkan metadata fallback untuk tampilan share artikel di media sosial ketika tidak memakai plugin SEO besar.

JSON-LD Schema

Schema Article/NewsArticle dan breadcrumb membantu struktur data artikel lebih jelas, dengan opsi menonaktifkan schema theme saat plugin SEO aktif.

Share buttons lengkap

Dukungan Native Share, Copy Link, WhatsApp, Telegram, Facebook, X/Twitter, LinkedIn, Pinterest, Line, Threads, Bluesky, Instagram, dan TikTok.

Table of Contents otomatis

Daftar isi dapat muncul otomatis ketika artikel memiliki heading yang cukup, sehingga artikel panjang lebih nyaman dibaca.

Related posts

Tampilan related posts bisa berupa list atau grid, lengkap dengan opsi thumbnail, snippet, metadata, dan kategori.

Ringan & modern

Tidak bergantung Bootstrap dan jQuery di frontend.

NewsPress memakai CSS Grid/Flexbox dan JavaScript native untuk menjaga frontend tetap ringan. Pengaturan performa disediakan sebagai opsi aman agar tetap fleksibel saat situs memakai plugin tambahan.

  • Disable WordPress emoji script/styles.
  • Disable wp-embed script.
  • Remove legacy head links.
  • Remove default block library CSS dan global styles jika tidak dibutuhkan.
  • Remove Dashicons untuk pengunjung yang belum login.
  • Preload homepage hero image dan featured image artikel.

Cara mulai

Instalasi singkat untuk pengguna WordPress.

Upload theme

Buka Appearance → Themes → Add New → Upload Theme, lalu upload file newspress.zip.

Aktifkan dan atur menu

Aktifkan NewsPress, buat menu di Appearance → Menus, lalu assign ke Primary Menu dan Footer Menu.

Sesuaikan Customizer

Atur logo, warna, layout, breaking news, trending, SEO, performa, dan slot iklan dari Appearance → Customize.

Bangun homepage

Gunakan Appearance → Homepage Builder dan Widgets untuk mengatur hero, row artikel, sidebar, slider, popular posts, dan area iklan.

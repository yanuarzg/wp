Fitur Unggulan

Satu theme untuk kebutuhan utama website berita.

NewsPress membawa fitur editorial yang biasanya butuh banyak plugin menjadi satu paket theme yang rapi, ringan, dan mudah diatur.

Homepage Builder

Atur hero, slider, grid, row kategori, latest post, dan widget area tanpa page builder berat.

Breaking News

Ticker berita terbaru berdasarkan recent post, kategori, atau tag breaking-news dengan label custom.

Trending Topic

Tampilkan trending berdasarkan tags atau categories, dengan rentang hari ini, 7 hari, 30 hari, atau all time.

Daftar Isi Otomatis

TOC otomatis dari heading H2-H6 dengan marker bertingkat, toggle, dan tampilan mobile friendly.

View Counter

Sistem view counter bawaan dengan rate limit untuk artikel, widget popular, dan statistik post.

Share Click Counter

Hitung klik share artikel dan tampilkan format seperti 125 Share di area share single post.

Dark Mode

Mode gelap dengan pengaturan background, primary, accent, dan text color langsung dari Customizer.

One Click Demo Import

Import demo posts, kategori, menu, widgets, Customizer, dan homepage settings dari default-import.json.

License & Auto Update

Sistem lisensi domain, grace period, kunci fitur premium, dan auto update theme via GitHub.

Mulai Lebih Cepat

Import demo, ganti branding, lalu mulai publish.

Distribusi Terkontrol

Lisensi domain dan auto update dari dashboard WordPress.

NewsPress dirancang agar distribusi theme lebih rapi: pembeli mendapat license key, fitur premium aktif berdasarkan domain, dan update theme bisa dikirim melalui GitHub.
